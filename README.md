# KKA
Topik: Open-World Sandbox
Judul: Mineria
Deskripsi:
Latar Belakang: dikarenakan kita ingin menggunakan ursina sebagai game engine, ursina itu sendiri merupakan game engine yang sepenuhnya berjalan di python, kami juga membuat game ini berdasarkan game favorit kami yaitu minecraft.
Algo: Informed Search, World-GAN, Uninformed Search
Cara kerja/Flowchart: User create new world -> program generate berdasarkan template -> mob dan player spawn -> hostile mob mendapatkan informasi terkait player dan menyerang -> friendly mob akan jalan bebas (Uninformed Search)
