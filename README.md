Latar Belakang: dikarenakan kita ingin menggunakan ursina sebagai game engine, ursina itu sendiri merupakan game engine yang sepenuhnya berjalan di python, kami juga membuat game ini berdasarkan game favorit kami yaitu Minecraft dan Teraria.
